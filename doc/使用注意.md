## 路径

1. 包内引用，用相对路径或别名

```js
// 例如：packages/base/src/index.ts 文件引用 packages/base/src/layout
import { AdminLayout, BasicLayout } from '@base/layout/index.ts';

// 或者
import { AdminLayout, BasicLayout } from './layout/index.ts';
```

2. 包外引用，用包名

```js
// 例如 /projects/vite-vue 文件引用 packages/base/src/router/index.ts
import router from '@lee/base/src/router/index';
```
