## @lee/theme

**功能**

使用 presets 中的主题色，配置多套主题（每套包含明暗色）

1. 配置多套主题（每套包含明暗色）
2. 配置naive-ui 适配多套主题
3. 配置间距，radius，字体大小等的变量文件
4. 将配置好的主题色，间距，radius，字体大小等的变量文件，生成 CSS 变量文件，并挂在到 dom 中
5. 提供切换主题的 API
6. 提供获取当前主题的 API
7. 提供切换明暗色的 API
8. 提供获取当前主题的 CSS 变量的 API
9. 提供初始设置主题的 API
10. 5,6,7,8，9 使用 hook 函数提供
