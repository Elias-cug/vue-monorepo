CSS 变量

  • var(--le-primary-1) → var(--le-primary-10)
  • var(--le-primary) = 主色（色阶 6）
  • var(--le-primary-hover) = 悬停色（色阶 5）
  • var(--le-primary-active) = 激活色（色阶 7）
  • var(--le-primary-suppl) = 补充色（色阶 4）

使用建议

  • 背景色 ：使用色阶 1-3，越浅越适合大面积使用
  • 交互元素 ：使用色阶 4-6，主色用于按钮、链接等
  • 文字和边框 ：使用色阶 7-10，确保足够的对比度
  • 状态表示 ：悬停用色阶 5，激活用色阶 7
  • 层次区分 ：相邻元素使用相邻色阶，避免跳跃
